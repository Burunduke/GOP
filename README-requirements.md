# Зависимости проекта GOP

Этот документ описывает различные варианты установки зависимостей для проекта GOP (Гиперспектральная обработка и анализ растений).

## Файлы зависимостей

### Основные файлы

- **`requirements.txt`** - Основные зависимости для работы проекта
- **`requirements-dev.txt`** - Зависимости для разработки
- **`requirements-gpu.txt`** - Зависимости с поддержкой GPU
- **`requirements-all.txt`** - Все возможные зависимости

## Варианты установки

### Базовая установка

Для базовой работы с проектом:

```bash
pip install -r requirements.txt
```

### Установка для разработки

Для разработки, тестирования и контрибуции в проект:

```bash
pip install -r requirements-dev.txt
```

### Установка с поддержкой GPU

Для ускорения вычислений с использованием GPU (только Linux/Windows):

```bash
pip install -r requirements-gpu.txt
```

**Примечание:** Для GPU поддержки требуется CUDA. Раскомментируйте соответствующие строки в `requirements-gpu.txt` в зависимости от вашей версии CUDA.

### Полная установка

Для установки всех возможных зависимостей:

```bash
pip install -r requirements-all.txt
```

## Установка через setup.py

Альтернативный способ установки через setuptools:

### Базовая установка
```bash
pip install .
```

### С дополнительными компонентами
```bash
# Для разработки
pip install .[dev]

# С поддержкой GUI
pip install .[gui]

# С поддержкой GPU
pip install .[gpu]

# С продвинутыми функциями
pip install .[advanced]

# С облачным хранением
pip install .[cloud]

# Все компоненты
pip install .[all]
```

## Требования к системе

### Минимальные требования
- Python 3.9+
- 8GB RAM
- 2GB свободного места на диске

### Рекомендуемые требования
- Python 3.10+
- 16GB RAM
- 5GB свободного места на диске
- GPU с поддержкой CUDA (для ускорения)

### Зависимости от ОС
- **Windows**: Требуется Visual Studio Build Tools для некоторых пакетов
- **Linux**: Требуются заголовочные файлы для GDAL
- **macOS**: Некоторые GPU зависимости недоступны

## Устранение проблем

### Проблемы с GDAL
```bash
# Ubuntu/Debian
sudo apt-get install gdal-bin libgdal-dev

# macOS
brew install gdal

# Windows
pip install GDAL-3.7.0-cp310-cp310-win_amd64.whl
```

### Проблемы с PyTorch
```bash
# CPU версия (рекомендуется для начала)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu

# GPU версия
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Проблемы с PyQt6
```bash
# Ubuntu/Debian
sudo apt-get install python3-pyqt6

# macOS
brew install pyqt6

# Windows
pip install PyQt6
```

## Версионирование

Проект использует семантическое версионирование зависимостей:
- Основные версии: `>=1.0.0,<2.0.0`
- Минорные версии: `>=1.5.0,<1.6.0`
- Патч-версии: `>=1.5.2,<1.6.0`

Это обеспечивает совместимость при обновлении и предотвращает проблемы с ломающими изменениями.

Для обновления до последних совместимых версий:

```bash

```

Для проверки устаревших зависимостей:

```bash
pip list --outdated
```
