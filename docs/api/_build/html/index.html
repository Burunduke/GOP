Обзор проекта

GOP (Гиперспектральная Обработка Растений) - это научно-ориентированная библиотека Python для:

  • Обработки гиперспектральных данных (BIL/HDR, TIFF)

  • Создания ортофотопланов

  • Расчета вегетационных индексов

  • Сегментации изображений сверхвысокого разрешения

  • Научного анализа и статистики

Основные возможности

  • Обработка гиперспектральных данных: Радиометрическая и атмосферная коррекция, шумоподавление, спектральная калибровка

  • Создание ортофотопланов: Интеграция с OpenDroneMap и GDAL

  • Вегетационные индексы: Расчет более 9 научных индексов для оценки состояния растений

  • Сегментация: Каскадный подход с использованием DeepLabV3+ и CascadePSP

  • Научный анализ: Статистический анализ, корреляционный анализ, пространственный анализ

Быстрый старт

from src.core.pipeline import Pipeline

# Создание пайплайна
pipeline = Pipeline()

# Обработка данных
results = pipeline.process(
    input_path='data/hyperspectral.bil',
    output_dir='results/',
    sensor_type='Hyperspectral'
)

# Получение результатов
print(f"Ортофотоплан: {results['orthophoto_path']}")
print(f"Состояние растений: {results['plant_condition']['classification']['class']}")
