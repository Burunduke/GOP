Введение в GOP
==============

GOP (Гиперспектральная Обработка Растений) - это научно-ориентированная библиотека Python, предназначенная для обработки гиперспектральных данных и анализа состояния растений с использованием вегетационных индексов.

Назначение библиотеки
---------------------

Библиотека разработана для научных исследований в области:

* Дистанционного зондирования Земли
* Точного земледелия
* Мониторинга состояния растительности
* Агрономических исследований
* Экологического мониторинга

Архитектура проекта
-------------------

Проект построен по модульному принципу с четким разделением ответственности:

Основные модули
~~~~~~~~~~~~~~~

**core** - Ядро системы
   * :class:`~src.core.pipeline.Pipeline` - Основной пайплайн обработки
   * :class:`~src.core.config.Config` - Управление конфигурацией

**processing** - Модули обработки данных
   * :class:`~src.processing.hyperspectral.HyperspectralProcessor` - Обработка гиперспектральных данных
   * :class:`~src.processing.orthophoto.OrthophotoProcessor` - Создание ортофотопланов

**indices** - Расчет вегетационных индексов
   * :class:`~src.indices.calculator.VegetationIndexCalculator` - Калькулятор индексов
   * :class:`~src.indices.definitions.IndexDefinitions` - Определения индексов

**segmentation** - Сегментация изображений
   * :class:`~src.segmentation.segmenter.ImageSegmenter` - Сегментатор изображений

**utils** - Вспомогательные утилиты
   * :class:`~src.utils.logger.setup_logger` - Настройка логирования
   * :class:`~src.utils.visualization.visualize_indices` - Визуализация результатов

Научная методология
-------------------

Библиотека реализует современные научные подходы к обработке гиперспектральных данных:

1. **Предварительная обработка**
   * Радиометрическая коррекция
   * Атмосферная коррекция
   * Шумоподавление (PCA, MNF, вейвлеты)
   * Спектральная калибровка

2. **Анализ данных**
   * Расчет вегетационных индексов
   * Статистический анализ
   * Корреляционный анализ
   * Пространственный анализ

3. **Визуализация**
   * Создание ортофотопланов
   * Визуализация индексов
   * Научные графики и диаграммы

Поддерживаемые форматы данных
-----------------------------

**Входные форматы:**
* BIL (Band Interleaved by Line)
* HDR (Header files)
* TIFF/GeoTIFF
* IMG

**Выходные форматы:**
* GeoTIFF
* JSON (результаты анализа)
* PNG/JPG (визуализация)

Требования к системе
--------------------

**Минимальные требования:**
* Python 3.9+
* 8 ГБ оперативной памяти
* 2 ГБ свободного дискового пространства

**Рекомендуемые требования:**
* Python 3.10+
* 16 ГБ оперативной памяти
* 10 ГБ свободного дискового пространства
* GPU с поддержкой CUDA (для ускорения обработки)
